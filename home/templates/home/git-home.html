git for Beginners


Git is a distributed version-control system for tracking changes in source code during software development. It is designed for coordinating work among programmers, but it can be used to track changes in any set of files. You will hear the word git and github a lot in programming because it is essential for working in groups. In this course you will learn about working with git and its different functions like push, pull, add, commit and amny more.
